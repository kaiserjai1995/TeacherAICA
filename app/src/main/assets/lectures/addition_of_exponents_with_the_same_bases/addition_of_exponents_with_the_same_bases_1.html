Here, we simplify an expression by using the product rule for adding exponents.

To develop a pattern for adding exponential expressions with the same base, we consider the product x2 ⋅ x3.

The expression x2 means that x is to be used as a factor two times and the expression x3 means that the x is to be used as a factor three times. So, we will have:

x2x3 = x ⋅ x (2 factors of x)   ⋅   x ⋅ x ⋅ x (3 factors of x)
= x ⋅ x ⋅ x ⋅ x ⋅ x (5 factors of x)
= x5
