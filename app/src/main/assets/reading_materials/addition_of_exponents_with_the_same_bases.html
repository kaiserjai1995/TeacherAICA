Addition of the Exponent with the Same Bases

Here, we simplify an expression by using the product rule for adding exponents.

To develop a pattern for adding exponential expressions with the same base, we consider the product x2 ⋅ x3. The expression x2 means that x is to be used as a factor two times and the expression x3 means that the x is to be used as a factor three times. So, we will have:

x2x3 = x ⋅ x (2 factors of x)   ⋅   x ⋅ x ⋅ x (3 factors of x)
= x ⋅ x ⋅ x ⋅ x ⋅ x (5 factors of x)
= x5

In general.

xmxn = x ⋅ x ⋅ x ′ ′ ′ ′ ⋅ x (m factors of x)   ⋅   x ⋅ x ⋅ x ′ ′ ′ ′ ⋅ x (n factors of x)
= x ⋅ x ⋅ x ⋅ x ⋅ x ⋅ ′ ′ ′ ′ ⋅ x ⋅ x ⋅ x (m + n factors of x)
= xm+n

If m and n are natural numbers, then xmxn = xm+n

Example: Simplify the following expressions:
a. x3x4 = x3+4   b. y2y4y   c. (2y3)(3y2)


SOLUTION

a. x3x4
= x3+4  
Add the two exponents together
= x7
3 + 4 = 7

b. y2y4y
= (y2y4)y
Use the associative property of multiplication to group y2 and y4 together
= y6y
Add the exponents of terms inside the parenthesis. 2 + 4 = 6
= y6+1
Add the exponents of y6 and y (y = y1)
= y7
6 + 1 = 7

c. (2y3)(3y2)
= (2)(3)(y3y2)
Use the commutative and associative properties of multiplication to group the coefficients and the variables together.
= 6y3+2
Multiply the coefficients then add the exponents
= 6y5
3 + 2 = 5

To multiply two exponential expressions with the same base, keep the base and add the exponents.
